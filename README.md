📌 College Campus Guide – Interactive Web Application
An interactive college campus tour and information system designed to help students, staff, and visitors easily explore and navigate the campus.

🚀 Features
📷 Camera & Image Upload – Capture a live photo using your device’s camera or upload an image, then send it to an API for recognition or processing.

🔍 Smart Search – Quickly find important campus locations like offices, lecture halls, and facilities using a live search filter.

🗺️ Integrated Campus Map – View the campus layout on Google Maps with key buildings and areas marked for easy navigation.

🎥 Video Tour – Watch an embedded campus video showcasing facilities and the environment.

🏛️ 360° Virtual Tours – Access Matterport 3D walkthroughs of library, labs, and sports areas.

Responsive Design – Optimized for both desktop and mobile devices with a clean, modern UI.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript

Media & Map Integration: Google Maps, Matterport 3D tours, video embedding

Camera API: Browser MediaDevices API for live photo capture

Search: Client-side JavaScript filter for instant results

🎯 Use Case
Perfect for college websites or orientation portals to give newcomers an immersive, easy-to-use guide to the campus — reducing confusion and improving the visitor experience.
